# WebexToLink

Copia negli appunti il link di una registrazione webex (m3u8).
In questo modo è possibile riprodurre una registrazione webex su qualsiasi player (come VLC) senza scaricare la lezione.

# Come installare l'estensione

L'estensione può esssere scaricata direttamente gli store dei browser:

CHROME/EDGE: [`chrome web store`](https://chrome.google.com/webstore/detail/webextolink/mjkllhbckabdnkolalpielbflkcmkaag?hl=it)

FIREFOX: [`add-on mozilla`](https://addons.mozilla.org/it/firefox/addon/webextolink/)


# Usare l'estensione

In una pagina webex basta avviare il player e premere l'icona dell'estensione. In questo modo copia il link della registrazione (in formato .m3u8) negli appunti.

Puoi incollare il link su qualsiasi player video che supporta la riproduzione di flussi di rete, come `VLC`.
